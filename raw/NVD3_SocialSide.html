This page explores the rendering of a horizontal bar chart in NVD3, based on survey response data from the Pew Research Center’s Internet & American Life Project, “The Social Side of the Internet”. Permission to use the survey data was granted by the Pew Research Center’s Internet Project December 2013.

The charts were created in R using rCharts, survey, and shiny packages, and summarize the types of social groups in which female and male respondents reported being active.

Getting nv.d3.js to run in WordPress

The usual solutions to running javascript in WordPress pages (template files , WP-D3, and Raw HTML ) do not currently work for nv.d3.js. I've found a solution initially created for inserting Google Charts into Wordpress that successfully renders the chart I want. As a result, with the exception of the div tag in the rChart template, all of the javascript used to create the chart has been inserted into the header of this page using the Per page add to head plugin.
